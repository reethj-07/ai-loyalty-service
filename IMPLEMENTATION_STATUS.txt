HOW TO USE THE AGENTIC SYSTEM
================================================================================

1. START THE BACKEND:
   ```bash
   cd /workspaces/ai-loyalty-service
   uvicorn app.main:app --reload --port 8000
   ```

2. START THE FRONTEND:
   ```bash
   cd frontend
   npm run dev
   ```

3. CONFIGURE ENVIRONMENT:
   Add to .env:
   ```
   ANTHROPIC_API_KEY=your_key_here  # Required for LLM reasoning
   ```

4. ACCESS AGENT CONSOLE:
   Navigate to: http://localhost:8080/agent-console

5. START AUTONOMOUS OPERATION:
   - Click "Start Autonomous Operation" button
   - Agent begins 15-minute analysis cycles
   - OR click "Analyze Now" for immediate analysis

6. MONITOR AGENT ACTIVITY:
   - Agent Console shows real-time status
   - View agent decisions and reasoning
   - See learnings accumulate over time

7. API USAGE:
   ```bash
   # Start agent
   curl -X POST http://localhost:8000/api/v1/agents/start

   # Check status
   curl http://localhost:8000/api/v1/agents/status

   # Trigger analysis
   curl -X POST http://localhost:8000/api/v1/agents/analyze/now
   ```
