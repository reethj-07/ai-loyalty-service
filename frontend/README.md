# Frontend - Asteria Growth Cloud

React + TypeScript frontend for the Asteria Growth Cloud platform.

---

## Prerequisites

- Node.js 18+
- npm

---

## Setup

```bash
cd frontend
npm install
```

Create `.env` in `frontend/`:

```bash
VITE_API_BASE_URL=http://localhost:8000
```

---

## Run Dev Server

```bash
npm run dev
```

The app runs at `http://localhost:8080` by default.

---

## Build

```bash
npm run build
```

---

## Tests

```bash
npm test
```

---

## Notes

- Backend API base is configured via `VITE_API_BASE_URL`.
- CSV import for members, transactions, and campaigns is available in the UI.
